# Company-Information-mail-Automation-System
# AI Email Auto-Reply Agent with Make.com & Gmail

An intelligent email automation workflow built with **Make.com**, **Gmail**, and an **AI Agent** that automatically reads incoming emails, generates context-aware responses, and sends professional HTML email replies.

## 🚀 Features

* 📥 Monitors Gmail for new unread emails
* 🤖 Uses an AI Agent to understand incoming messages
* 💬 Generates professional HTML email responses
* 📤 Automatically sends replies back to the original sender
* 🏢 Includes customizable company information via the AI system prompt
* ⚡ Fully automated workflow with no manual intervention

## 🛠️ Workflow

1. **Gmail Trigger**

   * Watches for new unread emails.
   * Marks processed emails as read.

2. **AI Agent**

   * Receives the email content.
   * Uses predefined company information and instructions.
   * Generates a professional HTML email response.

3. **Gmail Sender**

   * Sends the AI-generated reply to the original sender.
   * Preserves the original email subject by adding the `Reply:` prefix.

## 📦 Technologies

* Make.com
* Gmail API
* OpenAI AI Agent (via Make AI Provider)
* HTML Email Formatting

## ⚙️ Customization

The AI assistant can be easily customized by editing the system prompt to include:

* Company information
* Business hours
* Contact details
* Services offered
* FAQs
* Brand tone and writing style

This makes the workflow suitable for customer support, sales inquiries, internal help desks, and general business communication.
